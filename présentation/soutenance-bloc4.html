C2 : Architecture Algorithmique Hybride

ST-GRU-GNN (Spatial)

Propagation de la congestion sur un graphe de 1520 nœuds H3. Capturer les dépendances spatiales complexes.

Réentraînement Quotidien

XGBoost (Réactif)

Prédiction à H+1h basée sur les lags récents, la météo et les vacances. Focus sur la fiabilité locale.

Inférence toutes les 15 min

Gain CPU : -98% par rapport à un réentraînement permanent grâce au découplage.

C2 : Pipeline de Données Medallion

  • Bronze : Ingestion brute (JSONB) immuable de 8 sources (OGC, SIRI Lite, GBFS).
  • Silver : Nettoyage (psycopg2), déduplication et normalisation géo PostGIS.
  • Gold : Features ML-ready, vues matérialisées et stockage optimisé pour l'inférence.

C4 : Pipeline CI/CD & Qualité Logicielle

  • Intégration (GitHub Actions) : Linting (Ruff), Type Check (Mypy), et Security Audit (Bandit, Gitleaks).
  • Test PostGIS Natif : Utilisation de conteneurs de services pour des tests d'intégration sans mocks.
  • Déploiement (Makefile) : Processus reproductible rsync + restart systemd pour une stabilité maximale sur VPS.
  • Garde-fous : Rollback automatique via `make rollback-vps` en cas d'échec de santé applicative.
